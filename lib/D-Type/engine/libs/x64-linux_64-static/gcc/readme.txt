This folder contains D-Type Engines in the form of 64-bit static libraries for gcc 8.3 or higher for Linux (x64)

* dtypestd.o - D-Type Standard Engine
* dtypepwr.o - D-Type Power Engine
* dtypetxt.o - D-Type Text Engine
* dtypelay.a - D-Type Text Layout Extension (using HarfBuzz)
* dtypeaux.a - D-Type D-Type External Format Plugin

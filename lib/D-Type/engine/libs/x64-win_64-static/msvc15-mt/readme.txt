This folder contains D-Type Engines in the form of 64-bit multi-threaded static libraries for MS Visual C++ 2017 or higher for Windows (x64)

* dtypestd_lib-mt.lib - D-Type Standard Engine
* dtypepwr_lib-mt.lib - D-Type Power Engine
* dtypetxt_lib-mt.lib - D-Type Text Engine
* dtypelay_lib-mt.lib - D-Type Text Layout Extension (using HarfBuzz)
* dtypeaux_lib-mt.lib - D-Type D-Type External Format Plugin

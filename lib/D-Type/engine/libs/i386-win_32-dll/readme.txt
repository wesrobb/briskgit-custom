This folder contains D-Type Engines in the form of 32-bit dynamic link libraries for Windows (Intel 386)

* dtypestd.dll - D-Type Standard Engine
* dtypepwr.dll - D-Type Power Engine
* dtypetxt.dll - D-Type Text Engine
* dtypelay.dll - D-Type Text Layout Extension (using HarfBuzz)
* dtypeaux.dll - D-Type D-Type External Format Plugin

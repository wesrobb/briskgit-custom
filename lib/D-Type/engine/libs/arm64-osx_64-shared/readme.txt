This folder contains D-Type Engines in the form of 64-bit shared libraries for macOS arm64 (Apple Silicon)

* libdtypestd.8.0.1.dylib - D-Type Standard Engine
* libdtypepwr.8.0.1.dylib - D-Type Power Engine
* libdtypetxt.8.0.1.dylib - D-Type Text Engine
* libdtypelay.8.0.1.dylib - D-Type Text Layout Extension (using HarfBuzz)
* libdtypeaux.8.0.1.dylib - D-Type D-Type External Format Plugin

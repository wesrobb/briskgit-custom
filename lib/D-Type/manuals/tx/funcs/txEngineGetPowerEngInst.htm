txEngineGetPowerEngInst

void txEngineGetPowerEngInst(DT_TXENGINE engine, DT_PDENGINE* power_engine)

Short Description: Get associated D-Type Power Engine Instance

As mentioned earlier, each Text Engine has its own internal instance of D-Type Power Engine. The txEngineGetPowerEngInst function allows applications to get a handle of this instance.

Parameters

ParameterDescription

engine

Handle of the previously created Text Engine instance.

power_engine

Pointer to the handle of the D-Type Power Engine instance.

Comments

This function is useful for applications that need to access D-Type Power Engine functions directly. For example, applications can call txEngineGetPowerEngInst to obtain the Power Engine instance before rendering PowerDoc objects using D-Type Power Engine.

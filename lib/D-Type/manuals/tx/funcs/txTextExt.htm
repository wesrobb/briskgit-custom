txTextExt

void txTextExt(DT_TXDOC text_doc)

Short Description: Deinitialize and destroy text document

This function destroys a text document created by txTextIni, txTextIniViaStream, txTextIniViaBuffer or txTextIniViaPowerDoc. This function should be called when the text document is no longer needed and before its Text Engine instance is destroyed.

Parameters

ParameterDescription

text_doc

Handle of the associated text document instance.

txFree

void txFree(void* mem_addr)

Short Description: Free memory allocated by the Engine

The txFree function is used to release memory allocated by D-Type Text Engine. Any memory allocated by a call to one of D-Type Text Engine functions is owned by D-Type Text Engine and must be freed by txFree.

Parameters

ParameterDescription

mem_addr

Memory address at which the memory block to be released begins. This must be a valid pointer to a memory buffer that was previously allocated by one of D-Type Text Engine's memory allocation functions (e.g. txTextDraw).

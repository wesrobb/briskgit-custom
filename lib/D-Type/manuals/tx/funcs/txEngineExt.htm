txEngineExt

void txEngineExt(DT_TXENGINE engine)

Short Description: Deinitialize and destroy D-Type Text Engine

The txEngineExt function deinitializes and destroys the Text Engine instance created by txEngineIniViaStream. Since each Text Engine instance has its own internal instance of D-Type Power Engine, this function also deinitializes and destroys that instance. The txEngineExt function must be called before the application terminates for each instance of D-Type Text Engine.

As mentioned earlier, a Text Engine instance can be referenced by multiple text documents. This means that before a Text Engine instance is destroyed using this function, all of its text document instances must first be destroyed by calling the txTextExt function.

Parameters

ParameterDescription

engine

Handle of the Text Engine instance to be destroyed.

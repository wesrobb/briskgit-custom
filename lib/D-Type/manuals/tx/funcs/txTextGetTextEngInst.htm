txTextGetTextEngInst

void txTextGetTextEngInst(DT_TXDOC text_doc, DT_TXENGINE* engine)

Short Description: Get associated Text Engine Instance

As mentioned earlier, each text document is associated with one Text Engine instance. The txTextGetTextEngInst function allows applications to get a pointer to this Text Engine instance.

Parameters

ParameterDescription

text_doc

Handle of the text document instance.

engine

Pointer to the handle of the associated Text Engine instance.

D-Type Text Layout Extension License

D-Type Text Layout Extension is an optional extension designed to be used in conjunction with D-Type Font Engine.

Two versions of D-Type Text Layout Extension are available: HarfBuzz-based D-Type Text Layout Extension and ICU-based D-Type Text Layout Extension. Both versions have exactly the same binary interface and feature the same API. As of D-Type vesion 6.0, ICU-based D-Type Text Layout Extension is no longer distibuted (although it is still available to existing D-Type customers upon request).

HarfBuzz-based D-Type Text Layout Extension

HarfBuzz-based D-Type Text Layout Extension internally relies on HarfBuzz, an open source OpenType text shaping engine. HarfBuzz is licensed under the so-called "Old MIT" license. For details, see the HarfBuzz license agreement.

ICU-based D-Type Text Layout Extension

ICU-based D-Type Text Layout Extension internally relies on the ICU LayoutEngine, which is a component of the ICU (International Components for Unicode). ICU LayoutEngine is covered by the X open source license. The X open source license allows ICU to be incorporated into a wide variety of software projects using the GPL license. Because the X open source license is non-viral, ICU also can be incorporated into non-open source products. For details, see the ICU license agreement.

General
Description A font driver class to handle single-master and multiple-master fonts and font collections in D-Type format. A font driver class to handle single-master TrueType fonts and font collections and/or single-master OpenType fonts and font collections with TrueType outlines. A font driver class to handle single-master OpenType fonts (and font collections if they become available) with Type 2/CFF outlines as well as Bare CFF fonts. A font driver class to handle single-master and multiple-master fonts in Adobe Type 1 format. A font driver class to provide limited support for single-master Type 3 PostScript fonts. Only Type 3 fonts that obey D-Type's strict parsing rules are supported.

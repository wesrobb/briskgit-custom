Function Prototypes

D-Type Engine

Engine Construction and Destruction

  • dtEngineIniViaStream — Create and initialize D-Type Standard Engine - via stream
  • dtEngineExt — Deinitialize and destroy D-Type Standard Engine

Engine Info

  • dtEngineGetVersion — Get D-Type Standard Engine version

Output

Setting Up Output Parameters

  • dtOutputSetMDC — Set Output to memory surface (MDC)
  • dtOutputSetAsMDC — Set Output to memory surface (MDC) - deprecated form
  • dtOutputSetAsHDC — Set Output to Windows Device Context (HDC) - Windows only
  • dtOutputSetStyleAttribs — Set Output style (color, transparency and special effect such as shadow, pattern fill, emboss, emboldening etc.)
  • dtOutputSetStyleEffects — Set Output style (advanced version)

Getting Parameters

  • dtOutputGetStyleAttribs — Get Output style (color, transparency and special effect such as shadow, pattern fill, emboss, emboldening etc.)
  • dtOutputGetStyleEffects — Get Output style (advanced version)

  • dtRasterizerGetGrayscale — Get Rasterizer's alpha levels (grayscale)

  • dtTypesetterSetQuality — Set Typesetter's quality

  • dtTypesetterSetSizeSubscale — Set Typesetter's size subscale
  • dtTypesetterSetTypeAttribs — Set Typesetter's type
  • dtTypesetterSetTypeEffects — Set Typesetter's type (advanced version)
  • dtTypesetterSetTypeEffectsTransformed — Set Typesetter's type using an additional transformation matrix (advanced version)

Getting Parameters

  • dtTypesetterGetPositioning — Get Typesetter's positioning mode
  • dtTypesetterGetSubpixels — Get Typesetter's subpixel levels
  • dtTypesetterGetCacheCompression — Get Typesetter's bitmap cache compression
  • dtTypesetterGetCachePolicy — Get Typesetter's bitmap cache policy
  • dtTypesetterGetQuality — Get Typesetter's quality
  • dtTypesetterGetHintMin — Get minimum pixel size when Typesetter scales narrow horizontal and vertical glyph features in hinted fonts to small sizes
  • dtTypesetterGetSizeSubscale — Get Typesetter's size subscale
  • dtTypesetterGetTypeAttribs — Get Typesetter's current type
  • dtTypesetterGetTypeEffects — Get Typesetter's current type (advanced version)

Reinitialization/Reconfiguration

  • dtTypesetterRealloc — Reallocate Typesetter

Pattern Catalog

Getting Information

  • dtPatternGetNrOfGroups — Get number of pattern groups in Pattern Catalog
  • dtPatternGetGroupCount — Get number of pattern fills in a pattern group
  • dtPatternGetBitmap — Get pattern fill's bitmap data

Font Catalog

Finding Fonts

  • dtFontGetCounter — Get count of font slots in Font Catalog
  • dtFontTest — Test font slot and return its font index
  • dtFontFind — Find font index using unique font identification string
  • dtFontFindMMInstance — Find font index of a Multiple-Master font instance
  • dtFontFindMMInstancePlus — Find font index of a Multiple-Master font instance

Adding/Removing Fonts

  • dtFontAddViaStream — Add new font to Font Catalog via a single stream and return its font index
  • dtFontAddViaStreams — Add new font to Font Catalog via a set of streams and return its font index
  • dtFontAddAsMMInstance — Add new font as a Multiple-Master font instance and return its font index
  • dtFontAddAsMMInstancePlus — Add new font as a Multiple-Master font instance and return its font index
  • dtFontRemove — Remove font from Font Catalog
  • dtFontRemoveAll — Remove all fonts in Font Catalog

Activation/Deactivation

  • dtFontMakeActive — Activate font
  • dtFontMakeInactive — Deactivate font
  • dtFontMakeInactiveAll — Deactivate all fonts in Font Catalog

Output

  • dtFontSaveToStream — Save font to stream with optional glyph subsetting, in the D-Type format

Getting Information

  • dtFontGetSysStatus — Get font's system status
  • dtFontGetMMAxisInfo — Get number of axes, base designs and instances in font
  • dtFontGetMMInstData — Get default axis values for all named font instances in font
  • dtFontGetDim — Get number of base designs in font
  • dtFontGetStringValue — Get a string value from font
  • dtFontGetStringValues — Get string values from font in multiple languages
  • dtFontGetNumericValue — Get a numeric value from font
  • dtFontGetGlyphSubsts — Get glyph substitutions as defined in font
  • dtFontGetNrOfGlyphs — Get number of glyphs in font
  • dtFontGetMetrics — Get global font metrics
  • dtFontGetGlyphIndex — Get font dependent glyph index based on given character code

Special

  • dtFontSetErrorCallback — Instruct font drivers to send human-readable descriptions of errors encountered in bad fonts to your own function(s). Call this function before adding a new font to Font Catalog.

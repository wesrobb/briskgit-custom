C/C++ Functions

dtFontRemove

DT_SWORD dtFontRemove(DT_DTENGINE engine, DT_ID_SWORD font_index)

Short Description: Remove font from Font Catalog

This function permanently removes a font or font instance from the Font Catalog of the specified Standard Engine instance.

Parameters

ParameterDescription

engine

Handle of the previously created Standard Engine instance.

font_index

Font index of the font or font instance to be removed from the Font Catalog.

Return Value

If successful, the function returns 1. Otherwise, the function returns 0

Comments

An application can use the dtFontRemove function to permanently remove a font or font instance from the Font Catalog when it is no longer needed. Once the specified font or font instance is removed, the value of font_index becomes invalid and any future references to that index are not allowed. Your application must ensure that the removed font index is not referenced or used any more, otherwise unexpected results might occur.

Of course, any other fonts or font instances that are still in the Font Catalog will keep their unique font indexes after dtFontRemove returns.

Applications should use this function only when a font or font instance should be permanently removed from the Font Catalog. If there is a chance that the referenced font or font instance will be used again in the future, it is better to just make it inactive using the dtFontMakeInactive function.

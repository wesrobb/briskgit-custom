CDTRasterizerV8

Exit

void Exit()

This method deinitializes the CDTRasterizerV8 object which was initialized by a previous call to the Init method. This method must be called before the CDTRasterizerV8 object is destroyed. The CDTRasterizerV8 destructor calls this method automatically.

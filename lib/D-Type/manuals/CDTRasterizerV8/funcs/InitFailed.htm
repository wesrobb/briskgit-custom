CDTRasterizerV8

InitFailed

bool InitFailed()

This method tells wheter the full initialization of the CDTRasterizerV8 object failed.

Return Value

If the full initialization of the CDTRasterizerV8 object failed, the return value is true. Otherwise, the method returns false.
